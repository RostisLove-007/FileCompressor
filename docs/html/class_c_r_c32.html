Detailed Description

Инкрементальный и одноразовый вычислитель контрольной суммы CRC-32.

Класс позволяет либо накапливать контрольную сумму по частям данных методом update(), либо вычислить её сразу для всего буфера статическим методом compute(). Внутреннее состояние хранится в инвертированном виде (как того требует алгоритм CRC-32) и инвертируется обратно при вызове value().

inlinestaticnoexcept

Обновляет внешний (инвертированный) накопитель CRC новым блоком данных.

Используется для инкрементального вычисления CRC по частям, когда накопитель хранится вызывающим кодом (например, между чанками файла), а не внутри объекта CRC32. Перед первым вызовом накопитель должен быть инициализирован значением 0xFFFFFFFF.

Returns
Обновлённое инвертированное значение накопителя CRC.
