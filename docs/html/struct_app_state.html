Detailed Description

Глобальное состояние приложения: виджеты GTK и данные рабочего потока.

Хранит как ссылки на все используемые виджеты интерфейса, так и атомарные/потокобезопасные поля, через которые рабочий поток сжатия обменивается данными (прогресс, статус, признак завершения и ошибки) с основным потоком GTK, обновляющим интерфейс по таймеру.
